# Maalem

A client-side dynamic canvas application.

## Run Locally

To run the application locally (static assets only):

```bash
npm run dev
```

To run the application locally with full Serverless Function/API support:

```bash
npm run dev:vercel
```

## Deploy to Vercel

To deploy this project to Vercel:

1. Push your changes to your git repository (GitHub, GitLab, or Bitbucket).
2. Connect your repository to Vercel.
3. Configure the environment variable `GEMINI_API_KEY` in your Vercel Project Settings under the **Environment Variables** tab.
4. Vercel will automatically detect the static project and deploy it.
